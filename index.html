바다 유치원 서버 규칙

규칙을 지키지 않는 점.
규칙을 어겨 받은 불이익은 저희 관리진이 책임 지지 않습니다.
버튼을 누르시면 자세한 규칙을 볼 수 있습니다.

규칙에 어긋난 행동에 반박은 48시간 내로 항의 ( 즉 본인 처벌에 대한 항의 )

규칙에 허점을 이용 할시 강력한 제재

규칙에 없는 상황이 발생시 관리진 회의 하에 처리

인원 수 : 게임 통방은 5명/ 일반 통방 5명 이상으로 제한됩니다

채널: 채널에 맞게 사용 하지 않을시

도배 : 단타 5번 / 임티 5번 / 같은 말 5번

채팅창 : 예의 또는 심한 욕설 또는 패드립

불쾌감 : 통화방 내에서 상대에게 불쾌감을 주는 행동 또는 심한 욕설

수위성 발언 : 수위방이 아닌곳에서 수위가 있는 발언

들낙 : 음성 채널을 5번 이상 들낙 할시

타섭 언급 : 통방 내 또는 채팅창에서 언급

부적절한 통방 이름 : 수위가 있는 이름 & 비속어 & 저격 하는 이름

기본적 예의 : 초면에 반말 / 공격적인 말투 / 실례가 되는 질문 등

그 밖에 상황은 관리자가 판단 하여 경고를 지급 할수 있습니다

프사 : 4명이상 맞추지 않기

수위성 / 불쾌감 : 수위가 쎄거나 불쾌감을 줄수 있는 프사는 금지

소개글 : 자살 / 죽자 / 부정적 글은 삭제

기타 : 남을 저격 하는 상메 금지

남탓 : 게임 도중 남탓 / 비난 / 조롱 모두 포함

존중 : 서로간에 존중 하지 않거나 또는 심한 욕설 & 패드립

트롤 : 게임 도중 고의적으로 트롤 하는 행위

기타 : 게임방에서 인원수 제한을 지키지 않거나 불쾌감 있게 들낙 또는 소음 방해

뒷메 / 뒷방 : 갠디로 가서 서버 뿌리는 행위

타섭 / 광고 : 타섭 언급 / 광고 언급 등등

여미새 / 남미새 : 심각한 플러팅 & 여미새 / 남미새 /여왕벌도 포함

프로필 : 내소개에 타섭 링크 기재 상메에 타섭 홍보 글

기타 : 서버 인원수 빼갈시 차단

그 밖에 상황은 관리자가 판단하여 서버 밴을 할수 있습니다
